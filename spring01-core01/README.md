# 다형성 (스프링은 다형성을 극대화해서 이용)

역할 = 인터페이스

구현 = 인터페이스를 구현한 클래스, 구현 객체

다형성으로 인터페이스를 구현한 객체를 실행 시점에 유연하게 변경할 수 있다. (클라이언트에 영향을 주지 않는 변경 가능)

클라이언트를 변경하지 않고, 서버의 구현 기능을 유연하게 변경할 수 있다.
